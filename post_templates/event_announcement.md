# テンプレート：イベント告知

使用場面：トーナメント・例会・イベントの開催告知記事
目安文字数：600〜1,200字

---

## 記事タイトル（案）

```
【告知】{大会名} {開催年} 参加受付中！
```

または（例会向け）

```
{地域名}バックギャモン例会 {月}月のスケジュール
```

---

## 本文

```
## イベント概要

| 項目 | 内容 |
|------|------|
| 大会名 | {大会名} |
| 開催日 | {開催日} |
| 場所 | {会場名・住所 または オンライン} |
| 形式 | {シングルエリミネーション / ダブルエリミネーション / スイス等} |
| 参加費 | {参加費} |
| 対象 | {誰でも参加可 / 会員限定 / レーティング条件等} |

---

## エントリー方法

{エントリー先URL・方法・締切日を記載}

締切：{締切日時}

---

## 主催・問い合わせ

{主催団体名}
{問い合わせ先（URL・メール等）}

---

## 情報源

- {情報ソース名}：{URL}
```

---

## 投稿時のチェックリスト

- [ ] 開催日・場所の確認
- [ ] エントリー締切日の確認
- [ ] 参加費の確認
- [ ] エントリーURLが正しく機能するか確認
- [ ] 公式サイトへのリンクを含めているか
