# テンプレート：グッズ紹介

使用場面：新製品・セール・おすすめ用具の紹介記事
目安文字数：800〜2,000字

---

## 記事タイトル（案）

新製品紹介：
```
【新製品】{商品名} レビュー：{一言キャッチ}
```

セール告知：
```
【セール】{ショップ名}で{割引率}オフ開催中（〜{終了日}）
```

まとめ記事：
```
バックギャモン初心者におすすめの{カテゴリ名} {個数}選
```

---

## 本文（新製品・レビュー向け）

```
## 商品概要

| 項目 | 内容 |
|------|------|
| 商品名 | {商品名} |
| メーカー | {メーカー名} |
| 価格 | {価格（税込・送料別など）} |
| 購入先 | {ショップURL} |
| サイズ | {サイズ（ボードの場合）} |
| 素材 | {素材} |

---

## 特徴・ポイント

{商品の特徴を3〜5点で箇条書き}

- {ポイント1}
- {ポイント2}
- {ポイント3}

---

## こんな人におすすめ

- {対象者1（例：初心者で手頃な価格を求める方）}
- {対象者2}

---

## まとめ

{総評を2〜3文で記述}

---

## 情報源・購入先

- {ショップ名}：{URL}
```

---

## 本文（セール告知向け）

```
## セール概要

{ショップ名}にて、{セール名・内容}が開催中です。

| 項目 | 内容 |
|------|------|
| 期間 | {開始日} 〜 {終了日} |
| 割引 | {割引率 または 割引額} |
| 対象商品 | {全品 / 一部商品} |
| クーポンコード | {あれば記載、なければ削除} |

詳細・購入はこちら：{URL}
```

---

## 投稿時のチェックリスト

- [ ] 価格・セール情報が最新か確認
- [ ] セール終了日の確認
- [ ] 購入URLが正しく機能するか確認
- [ ] 自分で購入・確認できていない場合は「未確認」と明記
