# テンプレート：トーナメント速報

使用場面：大会結果が出た直後の速報記事
目安文字数：800〜1,500字

---

## 記事タイトル（案）

```
【速報】{大会名} 結果：{優勝者名}が優勝！
```

または

```
{大会名} {開催年} 結果まとめ
```

---

## 本文

```
## 大会概要

{大会名}が{開催日}に{開催地（国・都市）}で開催されました。
{参加人数}名が参加した今大会、{一言コメント（規模感・注目点など）}。

---

## 結果

| 順位 | 選手名 | 国籍 |
|------|--------|------|
| 優勝 | {優勝者名} | {国籍} |
| 準優勝 | {準優勝者名} | {国籍} |
| 3位 | {3位名} | {国籍} |

{日本人選手がいる場合↓}
### 日本人選手の結果
- {選手名}：{成績・コメント}

---

## 大会のハイライト

{印象的な試合・場面・エピソードを2〜3文で記述}

---

## 情報源

- {情報ソース名}：{URL}
```

---

## 投稿時のチェックリスト

- [ ] 選手名・スペルの確認
- [ ] 順位・結果の確認
- [ ] ソースURLの記載
- [ ] タイトルに大会名・年を含めているか
- [ ] 日本人選手の情報を見落としていないか
